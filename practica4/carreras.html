Carreras Itexsal

El Itexsal ofrece una cantidad muy amplia y variada en cuanto a las carreras que nos ofrece, entre algunas de las opciones que tenemos para elegir, esta: Aeronautica, Automotriz, Computacion, Diseño grafico, etc.

Aeronautica

El Itexsal es la unica institucion educativa que posee la carrera de mantenimiento aerounautico en toda latinoamerica, esta carrera es una de las especialidades en las que mas se inscriben, por no decir que la carrera a la que mas gente se inscriben, es una gran oportunidad poder estudiar esta carrera, ya que la institucion cuenta con mantenimiento aeronautico para practicas cuando sea necesario.

Comunicaciones

Comunicaciones es otra de las grandes opciones que Itexsal nos ofrece para poder estudiar, esta carrera es una muy buena opcion, cuentan con salones especiales para practicas que se deban realizar cuando sea necesario.

Diseño grafico

Otra carrera que nos ofrece el itexsal, es diseño grafico, que es una carrera general y una muy buena opcion para estudiar, cuentan con salones especiales para las practicas que se deban realizar tanto digital como las practicas que se deban hacer en fisico.

Computacion

Esta especialidad que nos brinda el exsal, es otra de las carreras generales, es decir que duran 2 años, estamos muy bien distribuidos con equipos que tienen lo que se requiere para realizar practicas, esta carrera es una de las mas solicitadas y una de las mas atractivas.

Gastronomia

Gatronomia es otra de las opciones que itexsal nos ofrece, es una carrera que tiene una duracion de 2 años y es una carrera bastante atractiva hacia el publico, para mucha gente es una opcion muy buena y que te puede abrir muchas puertas hacia una oportunidad de trabajo fuera del pais.
